# FindMyStuff
A lost and found web app for Bennett University students.
Built with React, TypeScript, Tailwind CSS, and Supabase.

## Features
- BU email restricted login
- Post lost and found items with images
- 30-day auto expiry
- Claim items
- My Posts management

## Tech Stack
- React + TypeScript + Vite
- Tailwind CSS
- Supabase (Auth, Database, Storage)
- Vercel (Deployment)
